# Node.js MongoDB Product CRUD Application

A full-stack product management application built with **Node.js, Express, MongoDB, EJS, and Tailwind CSS**.

This project provides a RESTful API for product management and dynamically displays products using EJS templates and Tailwind CSS.

---

## Features

- Create Product API
- Get All Products
- Get Product by ID
- Update Product
- Delete Product
- Duplicate product prevention
- Schema validation (minlength, required fields)
- MongoDB Atlas database
- Dynamic product cards using EJS
- Responsive UI with Tailwind CSS

---

## Tech Stack

Backend
- Node.js
- Express.js

Database
- MongoDB Atlas
- Mongoose

Frontend
- EJS Template Engine
- Tailwind CSS

Tools
- Postman (API testing)
- Nodemon

---

## Project Workflow

1. Built REST API using **Express.js**
2. Connected **MongoDB Atlas** database using **Mongoose**
3. Created **Product Schema with validations**
4. Tested APIs using **Postman**
5. Rendered product data dynamically using **EJS**
6. Styled UI using **Tailwind CSS**
7. Displayed products as responsive **product cards**
